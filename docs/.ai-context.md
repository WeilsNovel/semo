# semo 项目当前状态（整合对话维护）

> 整合对话每次会话开始时读取此文件 + DECISIONS.md + .ai-memory.md + TECH_DEBT.md 确认状态。
> 三端对话也读此文件了解项目全貌。

## 项目概览
- 代号：semo
- 定位：健康AI拍照识别 + 记录 + 打卡 小程序
- 启动日期：2026-06-29

## 技术栈
- 前端：Uni-app + Vue3 + TypeScript（`packages/mp`）
- 服务端：NestJS + TypeScript（`packages/server`）
- 契约层：`@semo/shared`（DTO/错误码/枚举/响应结构）
- AI 设计层：`@semo/ai`（Prompt 模板 + 配置占位）
- 仓库：Monorepo（pnpm workspace）

## 三对话分工
| 对话 | 负责包 | 职责 | 不可越界 |
|---|---|---|---|
| 前端 | `packages/mp` | UI/页面/状态/调REST | server/shared/ai |
| 服务端 | `packages/server`（含 `src/ai` 运行时） | 业务/DB/鉴权/REST实现/AI调用 | mp/shared/ai的prompts |
| API | `packages/shared` + `packages/ai` | 契约/AI Prompt设计/接口文档 | mp/server业务代码 |
